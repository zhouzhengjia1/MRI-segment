# Project 2：脑胶质瘤 MRI 自动分割

## 1. 数据模态与特征

不同的 MRI 模态主要用于观察不同的生理解剖结构：

| 模态    | 缩写全称示例 | 主要观察内容                             |
| :------ | :----------- | :--------------------------------------- |
| **T1n** | T1-weighted  | 基本解剖结构                             |
| **T1c** | T1-contrast  | 增强后图像，突出强化肿瘤区域(ET)         |
| **T2w** | T2-weighted  | 对水肿、液体更敏感                       |
| **T2f** | T2-FLAIR     | 抑制脑脊液后，更清楚地观察水肿和病灶(WT) |

## 2. 模型输入形状 (Input Shape)

- **2D 训练：** `[B, 4, H, W]`

## 3. 切分区域与标签界定

在评价和预测时，主要关注以下三个维度的肿瘤区域：

- **ET (Enhancing Tumor)**
  - **说明**：增强肿瘤区域。
  - **特点**：主要在 T1c (T1ce) 模态上比较明显。

- **TC (Tumor Core)**
  - **说明**：肿瘤核心区域。
  - **特点**：通常包括增强肿瘤 (ET) 以及坏死、非增强肿瘤等区域。

- **WT (Whole Tumor)**
  - **说明**：整个肿瘤区域。
  - **特点**：WT = 肿瘤核心 (TC) + 瘤周水肿区域。

### 区域范围关系

Whole Tumor (WT)
├── Edema 水肿区域
└── Tumor Core (TC)
├── NCR / NET 坏死或非增强核心
└── ET 增强肿瘤

### 标签定义

在 BraTS 经典标签里通常是：

0 = background
1 = NCR/NET，坏死/非增强肿瘤核心
2 = ED，水肿
4 = ET，增强肿瘤

因为脑胶质瘤不是一个单一区域，它包含多个部分：

水肿 edema（T2w 和 T2f 上比较明显）
坏死核心 necrotic core(T1n 上比较明显)
增强肿瘤 enhancing tumor(T1c 上比较明显)
